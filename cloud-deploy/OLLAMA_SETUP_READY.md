# 2. 创建 GPU 实例：

#    - 模板: PyTorch 2.1.0
#    - 磁盘: 80GB+
# 3. SSH 进去安装 Ollama:

curl -fsSL https://ollama.ai/install.sh | sh
ollama serve &
