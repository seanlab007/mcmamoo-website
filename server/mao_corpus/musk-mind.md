# 马斯克思想核心：第一性原理与工程哲学

## 1. 第一性原理 (First Principles Thinking)
- **定义**：剥离事物的表象，将其拆解到最基本的真理，然后从头开始推导。
- **对比**：类比思维（Reasoning by Analogy）是模仿他人，而第一性原理是物理学思维。
- **应用**：在 SpaceX，马斯克发现火箭原材料成本仅为售价的 2%，于是决定自己制造。

## 2. 五步工作法 (The Five-Step Process)
1. **质疑每一项要求**：无论谁提出的要求，都要将其视为建议。
2. **删减流程**：如果不需要增加，就尽量删减。
3. **简化与优化**：在删减之后再进行优化，不要优化本该不存在的东西。
4. **加快周期时间**：在完成前三步后，尽可能提高速度。
5. **自动化**：最后一步才是自动化。

## 3. 风险承担与快速迭代
- **拥抱失败**：如果事情没有失败，说明你创新得还不够。
- **快速反馈**：通过不断的测试和失败获取真实数据，而不是停留在理论推导。

## 4. 终极目标导向
- **多行星文明**：SpaceX 的存在是为了让生命多行星化。
- **可持续能源**：Tesla 的使命是加速世界向可持续能源的转变。

## 5. 极度紧迫感
- **时间轴压缩**：设定看似不可能的最后期限，以激发团队的最大潜力。
